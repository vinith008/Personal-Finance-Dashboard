# Personal-Finance-Dashboard


A simple and efficient Finance Tracking Application built using Python. This project helps users manage their daily expenses, track income, and maintain financial records easily.

---

## 🚀 Features

- Add Income & Expenses
- View Transaction History
- Calculate Total Balance
- Simple and User-Friendly Interface
- Data Storage (File/Database based)

---

## 🛠️ Technologies Used

- Python
- Django (if used)
- SQLite / MySQL (Database)
- HTML, CSS (Frontend - if applicable)

1. Clone the repository:
```bash

cd FinTrack-App

pip install -r requirements.txt

python app.py
